# ChurnIQ – Advanced Customer Churn Intelligence System

## 📌 Overview
ChurnIQ is a machine learning-based system that predicts customer churn and provides deep analytical insights using Explainable AI, Survival Analysis, and Uplift Modeling.

## 🚀 Features
- Churn Prediction using XGBoost & LightGBM
- Explainable AI (SHAP) for customer-level insights
- Survival Analysis to estimate time-to-churn
- Uplift Modeling to identify persuadable customers

## 🧠 Technologies Used
- Python
- Scikit-learn
- XGBoost
- LightGBM
- SHAP
- Lifelines
- Scikit-Uplift
